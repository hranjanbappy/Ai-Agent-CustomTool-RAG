Section 01

Research Findings

Every constant in the codebase is traceable to one of five peer-reviewed BUET or CDMP publications. No numbers come from the LLM.

🌏

Tectonic context: Bangladesh sits at the triple junction of the Indian, Eurasian, and Burma Sub-Plates. The Madhupur Fault beneath the DMDP area can generate M7.0–8.0 earthquakes. Classified as Seismic Zone II under BNBC 2020, Dhaka's true risk is severely understated due to deep Holocene alluvial deposits.

Section 02

Agentic Workflow

A LangGraph ReAct agent drives all query handling — it reasons about intent, selects the right tool, calls it, and formats the result. The LLM never invents numbers.

⚡

Core Principle: The LLM is never trusted with numbers. A LangGraph ReAct agent decides which tool to call. All vulnerability scores and cost estimates are produced by deterministic Python functions — every constant traceable to peer-reviewed BUET/CDMP literature. The LLM only formats the final report. Claude Code, Claude, and Gemini Pro were used to plan, code, test and fix bugs.
